Publish a site for freewith freenom and GitHub Pages

How to publish a website for free using freenom.com and GitHub Pages in 3 steps.

04/09/2015

Assumptions

Before I start, I'm going to assume a few things. Here they are:

  • You have got a freenom.com account (otherwise, do so for free now)
  • You have setup a free domain on freenom (choose one ending with .tk, .ml, .cf, .ga or .gq with over four letters before the . (dot) to register for free).
  • You have got a GitHub account (or are getting one for free from here)
  • You have setup a GitHub repository (can't help you here; it's up to you to make your site for the time being though I'm sure you can find some really good tutorials around on YouTube etc.)

Now, let's start.

Step 1: Get GitHub pages going

There's a good guide to doing so here.

Step 2: Get the DNS going

Once you've got GitHub pages going, head over to freenom.com, sign in and follow my lead.

  1. Go to Domain on the taskbar.
  2. Click on My Domains
  3. Press Manage Domain on the site you want to connect to GitHub pages
  4. Click on Manage Freenom DNS

Now, here comes the magical part:

Under Target, paste in the following DNS URL: 192.30.252.154.

Then press Enter or Return to get a new DNS area.

On the second DNS area you just trigger, under Target, paste in the following DNS URL: 192.30.252.153. This will mean that when someone types in www.<your site's URL>, they will get the right thing.

Step 3

Now, we go back to GitHub. On your repo, in the gh-pages branch you would have created back in Step 1. You now press the + button and put CNAME as the name of the file (without an extension). For the content of the file, you put in the URL of the site for which you had changed the DNS in Step 2. You now commit this file.

Loading ...

Once you've done this, have a little coffee, tea or spend 5 minutes doing something else. Once you've done that, go and open the site which you have been working with across this tutorial/article. You'll see that now you have the site you have been building all for free!

If not, you can contact me by email or DM me on Twitter.
